常用依赖管理POM.

##### Maven Dependency Management

```xml
<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>com.github.charlemaznable</groupId>
            <artifactId>complex-management</artifactId>
            <version>2023.2.2</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
```
